# BoardingHUB - Boarding House Tenant Management System

## 📋 **Project Overview**

BoardingHUB is a digital boarding house tenant management system designed specifically for small- to medium-scale Philippine boarding houses. Unlike public listing platforms, Boardify focuses exclusively on tenant tracking and rent management for landlords who already have established boarding houses.

### Key Concept: Property Code Enrollment
Similar to Google Classroom's class code system, Boardify uses unique property codes that landlords generate and share with their tenants. Tenants use these codes to connect to their specific boarding house property, eliminating the need for public listings and focusing purely on management of existing tenants.

## 🎯 Purpose
This repository contains the complete source code for the Boardify system, including:

- **Backend API** - Spring Boot REST API
- **Web Application** - React-based web interface
- **Mobile Application** - Native Android app (Kotlin)
- **Documentation** - Project documentation and setup guides

## ✨ Core Features

### For Landlords
- **Property Management** - Create and manage boarding house properties
- **Code Generation** - Generate unique enrollment codes for tenants
- **Tenant Roster** - View all enrolled tenants with room assignments
- **Rent Tracking** - Monitor payment statuses (current, pending, overdue)
- **Payment Records** - View complete payment history
- **Document Upload** - Store lease agreements and receipts
- **Dashboard Analytics** - Occupancy rates, collection summaries, overdue alerts
- **Email Notifications** - Send payment reminders to tenants

### For Tenants
- **Property Enrollment** - Join boarding house using landlord-provided code
- **Rent Dashboard** - View due dates and amounts
- **Payment History** - Complete record of past payments
- **Online Payments** - Pay rent via GCash/PayMaya (sandbox)
- **Document Access** - View/download lease agreements and receipts
- **Email Receipts** - Automatic payment confirmation emails
